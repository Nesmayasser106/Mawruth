# Mauruth – Folkloric Tourism Platform

Mauruth is a web-based platform focused on **folkloric tourism**, aiming to showcase local traditions, heritage, crafts, and cultural experiences.  
The project combines a clean front-end structure with a **Node.js API** to deliver interactive and dynamic features.

---

## Features
- User registration system
- Add folkloric experiences/items to the cart
- Remove items from the cart
- Dynamic cart management using APIs
- Organized and scalable project structure
- Cultural and heritage-focused content

---

## Technologies Used
- HTML
- CSS
- JavaScript
- Node.js
- RESTful APIs

## Functionalities Overview

User Registration: Allows users to create an account

Add to Cart: Users can add folkloric items or experiences

Remove from Cart: Users can delete items from the cart

API Integration: Backend APIs handle all cart operations

## Project Purpose

The goal of Mauruth is to promote authentic folkloric tourism while practicing full-stack development concepts such as API integration, user management, and modular project organization.
